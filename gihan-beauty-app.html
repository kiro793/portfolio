Gihan Beauty App

A native shopping app I designed and built from scratch for Gihan Beauty — React Native with Expo, talking live to the Shopify store through the Storefront API. One catalog, one cart, one brand — now in your pocket.

React Native / Expo Shopify Storefront API iOS & Android TypeScript

Why an app

The store, rebuilt as a native experience.

Gihan Beauty's customers reorder the same products often — an app makes that a two-tap habit instead of a browser search. The app mirrors the live store in real time: products, prices, collections, and stock all come straight from Shopify, so there's nothing to maintain twice.

Live catalog syncProducts, collections & prices via Storefront API — zero duplicate management
Cart & checkoutNative cart flowing into Shopify's secure checkout
Wishlist & ordersSign-in, order tracking, and saved favorites
Support built inWhatsApp, call, and email one tap away
Under the hood

Built like a product, not a demo.

Expo Router navigation

File-based routing with native stack transitions, a floating tab bar, and deep-link support.

GraphQL data layer

Typed Storefront API queries for products, collections, cart mutations, and customer accounts.

Brand-first UI

The store's visual identity translated to native patterns — clean cards, soft shadows, editorial banners.

Performance

Image caching with expo-image, lazy lists, and skeleton states for slow networks.

COD-ready flows

Checkout designed around Egyptian market reality: cash on delivery and WhatsApp confirmation.

EAS builds

Cloud builds and over-the-air updates — ship fixes without waiting for store review.
